Welcome to HostVirtualBingo.com

Prepare for the BINGO game

  1. Select Generate cards from the top links
  2. Select Add a player
  3. Type the player's name
  4. Type the number of bingo cards that player wants to play for each game
  5. To add additional players repeat steps 1-3 above
  6. Copy the URLs and send to the player next to the link

Play BINGO

  1. When you are ready to play select Host game from the top links
  2. To randomly draw a BINGO number, select Pick number from the Admin actions section
  3. The randomly selected number will be displayed in the Picked numbers section and highlighted
  4. When a player thinks they have a BINGO, ask for their card number
  5. Type the card number into the "Check Winner" section to check if the player has BINGO
  6. A display will be shown with the player's card, player's name and any possible BINGOs
  7. When you are ready to start a new game, select New game from the Admin actions section
  8. Repeat steps 2-7
